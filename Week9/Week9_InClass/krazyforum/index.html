KrazyForum API v1.0 - Documentation

read

Description
Returns a collection of posts in JSON format.

HTTP request
GET http://localhost/krazyforum/api/forum/read.php

search

Description
Returns a collection of posts matching the search query in JSON format.

HTTP request

  • Return all posts whose author is the specified author
    GET http://localhost/krazyforum/api/forum/search.php?a=brandon
    where a is the author parameter
    and the parameter value is case-insensitive (e.g. brandon, DARRYL, Yin kit, nICK).

  • Return all posts whose category is the specified category.
    GET http://localhost/krazyforum/api/forum/search.php?c=sports
    where c is the category parameter
    and the parameter value is case-insensitive (e.g. Love, sports, Pets, FOOD).

  • Return all posts whose author is the specified author AND category is the specified category.
    GET http://localhost/krazyforum/api/forum/search.php?a=darryl&c=love
    where:
          a is the author parameter
          AND
          c is the category parameter.
    Both parameter values are case-insensitive.

write

Description
Writes a comment into the forum.

HTTP request
GET http://localhost/krazyforum/api/forum/write.php

Required Parameters
Parameter Name Parameter Type Parameter Value
author String Sanchos
category String Food
comment String I love mala. I wish I had it everyday.
